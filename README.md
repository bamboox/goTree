# Installation
![goTree](/images/1.png)
## Go Get

```bash
$ go get github.com/bamboox/goTree.git
```
## Usage
```bash
goTree --directory /home/bamboo/Development/go/src/k8s.io/kubernetes/plugin 
```
OR
```bash
goTree --directory /home/bamboo/Development/go/src/k8s.io/kubernetes/plugin --filterStr .go
```
